## Core Features

<CardGrid stagger>
	<Card title="Concurrent Agents" icon="rocket">
		Run multiple LLM agents in isolated containers.
	</Card>
	<Card title="Multiple Runtimes" icon="laptop">
		Support for Docker, Apple Virtualization Framework, and Kubernetes.
	</Card>
	<Card title="Isolated Workspaces" icon="add-document">
		Git Worktrees ensure isolated code modification for each agent.
	</Card>
	<Card title="Hosted Architecture" icon="setting">
		Centralized state in a Hub with distributed execution on Runtime Hosts.
	</Card>
</CardGrid>

## How it Works

```d2
Hub: {shape: cloud}
RuntimeBroker: {shape: cylinder}
Agent: {shape: circle}

Hub -> RuntimeBroker: Orchestrates
RuntimeBroker -> Agent: Spawns
```
