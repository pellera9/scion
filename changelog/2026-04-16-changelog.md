# Release Notes (2026-04-16)

This update focuses on empowering grove members with improved agent management capabilities and updating core dependencies.

## 🚀 Features
* **Grove Management:** Introduced "Stop All" functionality for grove members. Owners and admins can now stop all agents within a grove, while regular members can stop their own agents. This action was previously restricted to platform admins.

## 📦 Dependencies
* **Security & Core:** Updated `dompurify` to 3.4.0 and `github.com/moby/spdystream` to 0.5.1.
